Email Checker Demo (Laravel5.4 and Vue 2.1)
=================


## Install

Clone the repo to your local machine

  ```
  git clone https://github.com/jcc/email-checker-demo.git
  cd email-checker-demo
  ```

Install the dependencies

  ```
  composer install
  npm install 
  ```

Compiling Assets

  ```
  npm run watch
  ```
